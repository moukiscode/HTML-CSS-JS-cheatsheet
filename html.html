What are HTML and XHTML ?

HTML stands for HyperTexte Markup Language, it is used to build the "skeleton" of a website. It allows to display various elements like text, images, videos and even forms.
You have to keep in mind that HTML is not the only markup languages and that it cannot do everything you want. For instance, PHP (Hypertext PrePrecessor) is a great way to make websites with accounts systems, but this language is harder to learn and to execute as it can't run on every computers.
